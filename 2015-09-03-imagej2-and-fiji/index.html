What is ImageJ2?

No one can tell you what ImageJ2 is. You have to see it for yourself.

An end-user application


/** Loads and displays a dataset using the ImageJ API. */
public void loadAndDisplayDataset(final File file) {
	// create the ImageJ application context with all available services
	final ImageJ ij = new ImageJ();

	// load the dataset
	final Dataset dataset = ij.scifio().datasetIO().open(file.getAbsolutePath());

	// display the dataset
	ij.ui().show(dataset);
}

Maintain backwards compatibility with ImageJ1

http://imagej.net/Compatibility

ImageJ2 is ImageJ 1.x, plus the new stuff
  • ImageJ2 includes ImageJ version 1.x
  • ImageJ2 user interfaces (UIs) are plugins
  • ImageJ2's LegacyUI = ImageJ 1.x as a plugin
  • IJ1 and IJ2 data structures are translated on the fly
  • "Mix and match" ImageJ 1.x and ImageJ2 capabilities
  • ImageJ 1.x features continue to work
  • New features of ImageJ 1.x developed by Wayne Rasband also work in this way
  • But ImageJ2 does depend on ImageJ 1.x—it also stands alone.
  • This design took many iterations to achieve!
